# Effective treatment allocation strategies under partial interference
This code accompanies the manuscript https://arxiv.org/abs/2504.07305, by Samantha G Dean, Georgia Papadogeorgou, and Laura Forastiere.

# Overview
Users can run model_sim.R and diffusion_sim.R to recreate the simulated results 
from the manuscript, or cai_application.R to recreate the application results 
from the manuscript. 

Other scripts, within the 'analysis_scripts' folder, create functions that are called within those scripts. 

model_sim.R and diffusion_sim.R are set up to be run parallel on a computing cluster. Sample code for sending these jobs to a cluster is included.
